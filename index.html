VERTASCAN — Documentation
Overview

VERTASCAN Surface Engine is a browser-based surfacing, shading and render-look tool built directly on top of Three.js/WebGL. It combines lightweight curve-to-surface modeling (revolve, extrude, loft) with a full PBR material system, HDRI/IBL lighting, and a postprocessing render pipeline for producing look-development stills from imported or hand-built geometry.

This is a surfacing & shading workbench, not a parametric CAD kernel — there is no BREP/NURBS solver and no history-based dependency graph. What is real: curve capture, revolve/extrude/loft surface generation, PBR materials, analysis shading, HDRI-based global illumination, a bloom/DOF/vignette postprocessing pipeline, import/export, and undo/redo.
Core Specification
Engine
Three.js r170 (WebGL2)
Renderer
ACES Filmic tone mapping, sRGB output
Shading Model
MeshPhysicalMaterial (metal/rough + clearcoat/transmission)
Lighting
Directional key + ambient + image-based lighting
Global Illumination
PMREM-filtered HDRI / equirect environment maps
Postprocessing
EffectComposer: Bloom, Bokeh DOF, Vignette + Grain
Units / Grid
Millimeters, 10mm grid spacing
Selection
Raycast picking + Translate/Rotate/Scale gizmo
Supported Formats
CategoryFormats
Model Import.glb / .gltf, .obj, .stl
Model Export.glb (binary glTF), .stl (ASCII)
Environment Import (IBL).hdr (Radiance HDRI), .jpg / .png equirectangular
Still Render Export.png (lossless), .jpg (quality 0.95)
Render ResolutionsViewport, 1280×720, 1920×1080, 2560×1440, 3840×2160
Feature Set
  • Sketch: ground-plane curve drawing (Catmull-Rom) with live preview
  • Surfacing: Revolve (lathe), Extrude (with bevel), Loft (2-curve ruled surface)
  • Primitives: box, sphere, cylinder, cone, torus
  • Mesh Tools: flip normals / winding, mirror on X
  • Materials: 7 PBR presets (carpaint, chrome, aluminum, glass, plastic, carbon) plus full per-object shader controls — metalness, roughness, clearcoat, clearcoat roughness, transmission, IOR, emissive intensity/color, opacity
  • Analysis Shading: Shaded PBR, Wireframe, Normal Map, Zebra Stripes, Curvature Tint
  • Lighting & Atmosphere: key light intensity/color/azimuth/elevation, ambient level, fog (color/near/far), exposure, backdrop color
  • Environment / IBL: HDRI & equirect world-map import, background toggle, environment intensity, background blur, one-click studio reset
  • Post Processing: Bloom (strength/radius/threshold), Depth of Field (focus/aperture/blur), Vignette + film grain
  • Scene Management: explorer list, visibility toggling, deletion, undo/redo, 4 camera bookmarks

Getting Started
  • Orbit: left-drag in the viewport
  • Pan: right-drag (or middle-drag) in the viewport
  • Zoom: scroll wheel
  • Select: click an object; click empty space or press Esc to deselect
  • Drag & drop a .glb / .obj / .stl file straight onto the viewport to import it
Sketch & Curves

Select Draw Curve in the left dock, then click points on the ground plane to place a curve. Press Enter or double-click to finish, or Esc to cancel. The finished curve appears in the Scene Explorer and can be fed into any Surfacing operation below.

Surfacing Operations
ToolHow to use
Revolve CurveSelect a curve, click Revolve Curve. The curve's XZ radius is swept 360° around the Y axis (lathe).
Extrude CurveDraw a closed-ish curve with 3+ points, select it, click Extrude Curve to pull it into a solid with beveled edges.
Loft (2 curves)Draw two curves. Select the first and click Loft, then select the second and click Loft again — a ruled surface is built between them.
Primitives & Mesh Tools

Box / Sphere / Cylinder / Cone / Torus drop a ready-made mesh into the scene. Flip Normals reverses winding + normals on the selected mesh (useful after an extrude/loft looks inside-out). Mirror X duplicates the selection, flipped across the X axis.

Transform & Keyboard Shortcuts
KeyAction
WMove (translate) gizmo
ERotate gizmo
RScale gizmo
DeleteDelete selected object
Ctrl/Cmd + ZUndo
Ctrl/Cmd + YRedo
EscDeselect / cancel current curve draw
EnterFinish the curve currently being drawn

You can also type exact Position / Rotation / Scale values in the Inspector's numeric fields, and jump the camera to Persp / Top / Front / Right bookmarks from the topbar.

Materials & Inspector

Select an object, then in the Scene tab's Inspector: click a swatch to apply a preset PBR material, use the color picker for a quick tint, or dial in the full shader — Metal, Rough, Clearcoat, Clearcoat Rough, Transmit, IOR, Emit intensity/color and Opacity — for precise, physically based looks (e.g. push Transmit + IOR ≈1.5 for glass, or Metal to 1 + low Rough for chrome).

Render Tab — Lighting & Atmosphere

Switch the right panel to Render. Under Lighting, adjust the key light's intensity, color and Azimuth/Elevation (its position on the sky dome), plus overall Ambient fill. Under Atmosphere, toggle Fog and tune its color/near/far falloff, set the render Exposure, and pick a Backdrop color for scenes without an environment image.

Render Tab — Environment / IBL

Import HDRI (.hdr) or Import World Map (jpg/png) to light the whole scene from a real environment — reflections and ambient light are generated automatically via PMREM. Enable Use as Scene Background to also show the map behind your model, and use Intensity / Bg Blur to balance it. Reset to Studio Env returns to the default neutral studio lighting.

Render Tab — Post Processing
  • Bloom: soft glow on bright areas — Strength for intensity, Radius for spread, Threshold for how bright a pixel must be before it blooms
  • Depth of Field: camera-style focus falloff — Focus sets the in-focus distance, Aperture and Max Blur control how strong the out-of-focus blur is
  • Vignette / Grain: subtle edge darkening and film grain for a finished, photographic render look
Analysis Shading Modes
ModeUse
Shaded PBRFull material + lighting preview (default)
WireframeInspect topology and surface density
Normal MapVisualize surface normal direction per-pixel
Zebra StripesReflection-line analysis for surface continuity/kinks
Curvature TintFacing-ratio false-color for quick curvature reads
Import / Export
  • Import — topbar Import button or drag-and-drop: .glb/.gltf, .obj, .stl
  • Export Model — topbar Export GLB / Export STL saves the full scene
  • Export Render — Render tab → Render Output: pick a resolution, then Export PNG or Export JPG to save a still image of the current camera view with all lighting and post effects baked in
Tips
  • Material edits always apply to the object's real material, even while an analysis mode like Normal or Zebra is active on screen — switch back to Shaded PBR to see them.
  • High Aperture/Max Blur values in Depth of Field exaggerate the effect — start small.
  • Rendering at 4K takes a moment longer; the viewport briefly resizes internally, then restores.

  • Export Model — topbar Export GLB / Export STL saves the full scene
  • Export Render — Render tab → Render Output: pick a resolution, then Export PNG or Export JPG to save a still image of the current camera view with all lighting and post effects baked in
Tips
  • Material edits always apply to the object's real material, even while an analysis mode like Normal or Zebra is active on screen — switch back to Shaded PBR to see them.
  • High Aperture/Max Blur values in Depth of Field exaggerate the effect — start small.
  • Rendering at 4K takes a moment longer; the viewport briefly resizes internally, then restores.
